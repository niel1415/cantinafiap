# CantinaFIAP — Sistema de Pedidos na Cantina

## 📌 Descrição do Problema
No dia a dia da FIAP, principalmente nos horários de intervalo e troca de aula, a cantina costuma apresentar filas grandes e atendimento sobrecarregado. Isso gera demora para realizar pedidos, dificuldade de organização para os funcionários e perda de tempo para os alunos, que normalmente possuem poucos minutos disponíveis.

Além disso, quando muitos pedidos são feitos ao mesmo tempo, podem ocorrer confusões na ordem de preparo, erros na entrega e falta de clareza sobre o status do pedido. Dessa forma, o processo atual acaba sendo menos eficiente do que poderia ser.

## 🚀 Solução Proposta
A solução proposta é o **CantinaFIAP**, um sistema de pedidos na cantina desenvolvido em Python via terminal, simulando o núcleo funcional de um aplicativo de autoatendimento.

Com ele, o aluno pode visualizar o cardápio, adicionar itens ao carrinho, finalizar pedidos, acompanhar o status e cancelar quando permitido. Já o atendente consegue visualizar todos os pedidos e atualizar seus status, melhorando a organização e reduzindo parte dos problemas causados pelas filas.

## 🛠 Tecnologias Utilizadas
- Python 3.11
- JSON para persistência de dados
- Terminal/CLI
